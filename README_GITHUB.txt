KC MarktKasse – GitHub-Publish

Nur den INHALT dieses publish-Ordners in das GitHub-Repository ziehen.
Danach GitHub Pages auf Branch main und Ordner / (root) einstellen.

Der Ordner enthält ausschließlich die Browser-Laufzeit der Kasse:
- index.html
- app.js
- styles.css
- manifest.webmanifest
- service-worker.js
- shared/runtime-flags.js
- assets (Artikelbilder und Logo)
- .nojekyll
